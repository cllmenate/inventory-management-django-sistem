# Guia de Contribuição

Obrigado pelo interesse em contribuir com o Sistema de Gestão de Estoque!

## Como Contribuir

1. Faça um Fork do projeto.
2. Crie uma branch para sua feature (`git checkout -b feature/minha-feature`).
3. Commit suas mudanças (`git commit -m 'Adiciona minha feature'`).
4. Push para a branch (`git push origin feature/minha-feature`).
5. Abra um Pull Request.

## Padrões de Código

Seguimos as diretrizes da PEP 8 e utilizamos as seguintes ferramentas:

- **Ruff**: Para linting e formatação.
- **MyPy**: Para checagem de tipos estática.
- **Pytest**: Para testes automatizados.
